# NGA 快速浏览总结热帖

## 功能

浏览 NGA (bbs.nga.cn) 中的 "网事杂谈"、"手机 网页游戏综合讨论"这两个模块，浏览其中的帖子名和高回复帖子内容，总结今日和最近的论坛热点并给出近几日论坛讨论动向，并给出部分核心论点的帖子链接。

## 实现方法

- 通过cookies方式登录

- 通过 LLM API 来调用 LLM 对帖子进行总结

- 有需要可以在本地记录一些浏览内容

- 在运行后，每天固定时间进行一次总结，存为一个 markdown 文件，同时还要支持用户手动触发浏览总结功能
